TechLab Ecommerce
Proyecto básico de gestión de productos desarrollado en Java como parte del curso de Back-End.
Permite agregar productos, actualizarlos y validar sus datos antes de guardarlos.

Funcionalidades
Alta de productos
Actualización de productos
Validaciones inmediatas:
nombre obligatorio, precio mayor que 0, stock no negativo, descripción y categoría válidas.
